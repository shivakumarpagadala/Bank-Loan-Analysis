
# Bank Loan Analysis – End-to-End Project
**Tools Used:** Microsoft SQL Server | Power BI | Python | Excel | Tableau

## Project Overview
This is an end-to-end data analytics project designed to analyze bank loan performance, borrower trends, and portfolio health. The project uses a full data pipeline — from raw data cleaning to SQL processing, visualization, and dashboard creation.

The system provides insights that help banks monitor loan applications, repayment behavior, borrower risk, and overall loan portfolio performance.

## Tech Stack
### Microsoft SQL Server
- Data Import & Storage
- Data Cleaning and Pre-Processing
- KPI queries

### Python
- Data cleaning & transformation
- EDA & visualization
- Dataset export for BI dashboards

### Excel
- Data validation & cleanup
- Pivot summaries

### Power BI
- Three dashboards (Summary, Overview, Details)
- KPI cards, maps, donut charts, trend analysis

### Tableau
- Additional visualization layer
- Interactive dashboards
